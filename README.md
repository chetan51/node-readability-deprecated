## node-readability ##

This is a port of [Arc90's Readability](http://lab.arc90.com/experiments/readability/) to [Node](http://nodejs.org/).

### Dependencies ###

Requires:

	*	Node.js
	*	[jsdom](https://github.com/tmpvar/jsdom).
